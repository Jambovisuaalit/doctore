# Tennis required inputs

- tournament, level, round, date, venue, and stable player identifiers;
- surface, indoor/outdoor status, altitude when modeled, and best-of format;
- current match status and official start time;
- book retirement, walkover, and void rules;
- model version and target market;
- recent match load, travel, injury/fitness status, and feature cutoff only when the production model uses them;
- exact line and period for set, games, totals, or props.
